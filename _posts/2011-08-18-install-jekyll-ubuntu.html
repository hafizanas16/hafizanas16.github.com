Jekyll adalah salah satu _gem_ yang bisa dibilang mesin blog statis. Jekyll dibuat dengan bahasa ruby dan merupakan mesin di balik Github Pages. Jekyll meng-generate file HTML dari tulisan yang dibuat dengan syntax Textile, Markdown, atau plain HTML. Mungkin banyak yang bertanya-tanya, "mengapa susuah-susah pake statis HTML, kalo banyak CMS bagus di internet?" Berikut adalah alasan saya menggunakan statis HTML.

  • Ringan
  • Mudah dan simple
  • Tidak perlu login halaman admin
  • Tidak perlu update keamanan, karena tidak ada yang bisa di-hack :D
  • Tidak perlu database. Semua post disimpan bentuk file

Instalasi Jekyll

Jekyll membutuhkan library Ruby agar bisa berjalan. Saya menggunakan Ubuntu 10.10 untuk lebih mudah, karena ribet kalau di Windows.

  1. Instal Ruby

    Instal library yang dibutuhkan saat instalasi Ruby: {% highlight bash %}$ sudo apt-get install build-essential libssl-dev libreadline5 libreadline5-dev zlib1g zlib1g-dev{% endhighlight %}

    Install ruby dari source (http://www.ruby-lang.org): {% highlight bash %} $ mkdir ~/src && cd ~/src $ wget http://ftp.ruby-lang.org/pub/ruby/1.9/ruby-1.9.2-p136.tar.gz $ tar -zxvf ruby-1.9.2-p136.tar.gz $ cd ruby-1.9.2-p136/ $ ./configure && make && sudo make install {% endhighlight %}

    Cek Ruby yang terinstal: {% highlight bash %} $ ruby --version {% endhighlight %}

    Jika instalasi berhasil akan muncul: {% highlight bash %} ruby 1.9.2p136 (2010-12-25 revision 30365) [x86_64-linux] {% endhighlight %}

  2. Update RubyGems

    Ruby menggunakan sebuah paket manajemen sistem yang dikenal dengan nama RubyGems, yang digunakan untuk mengatur instalasi dan maintenance program Ruby dan library-library-nya. Lakukan update RubyGems dengan perintah berikut: {% highlight bash %} $ sudo gem update --system {% endhighlight %}

  3. Install Jekyll

    Untuk menginstall Jekyll sangat mudah, hanyasatu baris syntax saja: {% highlight bash %} $ sudo gem install jekyll {% endhighlight %} Tunggu sampai download library yang dibutuhkan selesai.
